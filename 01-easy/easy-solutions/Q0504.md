## Q0504: [Base 7](https://leetcode.com/problems/base-7/)

```
Difficulty Level: EASY
```

```
Description:

Given an integer num, return a string of its base 7 representation.
```

#### Python3:

```
class Solution:
    def convertToBase7(self, num: int) -> str:
        n, s = abs(num), ''

        while n:
            n, remain = divmod(n,7)
            s = str(remain) + s

        return "-" * (num < 0) + s or "0"
```
