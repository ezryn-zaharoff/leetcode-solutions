## Q0019: [Remove Nth Node From End of List](https://leetcode.com/problems/remove-nth-node-from-end-of-list/)

```
Difficulty Level: MEDIUM
```

```
Description:

Given the head of a linked list, remove the nth node from the end of the list and return its head.
```

#### Python3:

```
class Solution:
    def removeNthFromEnd(self, head: List[str], n: int) -> List[str]:
        m = len(head) - 1

        if n > m:
            ans = []
        else:
            ans = [element for i, element in enumerate(head) if i != n]

        return ans
```
