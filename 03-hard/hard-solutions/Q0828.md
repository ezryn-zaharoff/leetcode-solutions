## Q0828: Count Unique Characters of All Substrings of a Given String

```
Difficulty Level: HARD
```

```
Description:

Let's define a function countUniqueChars(s) that returns the number of unique characters in s.

â€¢ For example, calling countUniqueChars(s) if s = "LEETCODE" then "L", "T", "C", "O", "D" are the unique characters since they appear only once in s, therefore countUniqueChars(s) = 5.

Given a string s, return the sum of countUniqueChars(t) where t is a substring of s. The test cases are generated such that the answer fits in a 32-bit integer.

Notice that some substrings can be repeated so in this case you have to count the repeated ones too.
```

#### Python3:

```
class Solution:
    def uniqueLetterString(self, s: str) -> int:
        char_idx = defaultdict(list)
        for i, char in enumerate(s):
            char_idx[char].append(i)
        
        ans = 0
        for char in char_idx:
            sub = [-1] + char_idx[char] + [len(s)]
            for i in range(1, len(sub)-1):
                ans += (sub[i] - sub[i-1]) * (sub[i+1] - sub[i])
        
        return ans
```
